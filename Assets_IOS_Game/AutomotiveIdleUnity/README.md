# Automotive Idle Unity Prototype

Open this folder in Unity:

```text
/Users/leozmoos/Downloads/Assets IOS Game/AutomotiveIdleUnity
```

Main scene:

```text
Assets/Scenes/RoadsideRepairGarage.unity
```

The scene contains a generated roadside repair garage prototype inside a small low-poly city with:

- road and driveway layout;
- surrounding city road grid;
- commercial storefront block;
- suburban houses and driveways;
- industrial warehouse/factory area;
- parked and passing traffic;
- street lights, trees, fences, signs, and pedestrians;
- garage shell and reception area;
- player, customer, employee preview, and vehicle models;
- basic repair bay;
- parts source;
- payment zone;
- tire station, oil station, queue slot, and employee hire construction pads;
- labeled colored gameplay zones;
- vehicle waypoint markers;
- isometric camera and lighting.

If you want to regenerate the scene from the blueprint script, use Unity's top menu:

```text
Automotive Idle > Build Roadside Garage Scene
```

The generated script is here:

```text
Assets/Editor/BuildRoadsideGarageScene.cs
```

The detailed design and asset catalog is also included:

```text
Assets/automotive_idle_scene_asset_catalog.md
```
